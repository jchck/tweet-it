# Tweet It

A Wordpress plugin to make it easier for your readers to tweet your content

## Use

Just wrap whatever text you'd like to easily be tweeted out with a `[tweet-it]` shortcode.

`[tweet-it]Just like this.[/tweet-it]`

Pretty easy, right?
